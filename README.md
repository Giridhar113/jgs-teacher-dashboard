# JGS Teacher Dashboard

Teacher dashboard for the **JGS College Website Management System**.

This repository contains only the teacher-facing frontend used by faculty to manage attendance, marks, assignments, leave approvals, class analytics, PTM bookings, and student doubts.

## Project Scope

This repo is only for the teacher dashboard.

It does not include:

- Public website
- Admin dashboard
- Student dashboard
- Parent dashboard
- Shared backend API source code
- MongoDB credentials
- Local `.env` files

## Main Features

- Teacher login
- Faculty dashboard overview
- Attendance management
- Bulk attendance CSV import
- Gradebook
- Marks entry
- Assignment grading
- Student doubt replies
- Leave approval
- Class analytics
- Faculty timetable
- Parent messages
- PTM bookings
- Notices
- HR/payroll view

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- Papa Parse
- jsPDF
